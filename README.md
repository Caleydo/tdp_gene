=====================

This repository provides a collection of TargID views and visualization that could be used independent from entry point.

Installation
------------

```
git clone https://github.com/phovea/targid_common.git
cd targid_common
npm install
```

Testing
-------

```
npm run test
```

Building
--------

```
npm run build
```
